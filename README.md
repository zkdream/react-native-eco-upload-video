# react-native-eco-upload-video

ECO视频分片上传

## Installation


```sh
npm install react-native-eco-upload-video
```


## Usage


```js
 import { uploadVideoWithUrl } from 'react-native-eco-upload-video';

// ...
```
通过来分片上传uploadVideoWithUrl视频到到服务器
